# Antigravity Japanese Patch (Antigravity_Jpn_Patch)

このプロジェクトは、Google DeepMindが開発するAIエージェントプラットフォーム「Antigravity」のUIを日本語化するための非公式パッチです。

## 動作確認済みバージョン
- **Antigravity v2.0.11**
- （※今後のアップデートにより、仕様変更で適用できなくなる可能性があります）

## 特徴
- **自動適用ツール**: `install.ps1` を実行するだけで、ワンクリック（ワンコマンド）で誰でも簡単に日本語化が可能です。
- **外部辞書ファイル**: 翻訳辞書データは `japanese_patch.json` で管理されています。このJSONを更新して再度インストールスクリプトを実行するだけで、常に最新の翻訳を反映できます。
- **プラグインとスキルの翻訳対応**: UIの基本テキストだけでなく、MCPサーバーや各種スキルの説明文の日本語化にも対応しています。

## インストール方法

1. このリポジトリをZIPでダウンロード（「Code」ボタン → 「Download ZIP」）して解凍するか、`git clone` します。
2. 解凍したフォルダの中にある **`install.bat`** をダブルクリックして起動します。
   （※ 実行ポリシーの制限などを自動で迂回して、日本語のコンソール画面が立ち上がります）
3. 実行中に「現在のクリーンな状態でバックアップファイルを上書きしますか？」と聞かれた場合は、**`n`** を入力してEnterキーを押してください。
   （※Antigravity本体を公式アップデートした直後の場合のみ **`y`** を入力して上書きしてください）
4. パッチの適用処理が正常に完了したら、Antigravity アプリを（再）起動してください。

> 💡 必要なツール：Node.js がインストールされている必要があります。

## 元の英語版に戻す方法
**`uninstall.ps1`** を右クリックして「PowerShell で実行」を選択してください。自動的にバックアップから元の状態に復元されます。

## 翻訳の追加・修正
1. `japanese_patch.json` をテキストエディタで開き、翻訳を追加・修正します。
2. 再度 `install.bat` をダブルクリックして実行すると、変更がアプリに反映されます（この時のバックアップ上書き確認は **`n`** を選択してください）。

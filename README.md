# Antigravity Japanese Patch (Antigravity_Jpn_Patch)

このプロジェクトは、Google DeepMindが開発するAIエージェントプラットフォーム「Antigravity」のUIを日本語化するための非公式パッチです。

## 特徴
- **動的テキスト置換**: アプリ本体のソースコードを極力書き換えずに、`preload.js` でのDOM監視を利用して英語UIを動的に日本語に置き換えます。
- **外部辞書ファイル**: 翻訳辞書データは `japanese_patch.json` に外部化されているため、アプリ本体を再コンパイルすることなく簡単に翻訳を追加・修正できます。
- **プラグインとスキルの翻訳対応**: UIの基本テキストだけでなく、MCPサーバーや各種スキルの説明文の日本語化にも対応しています。

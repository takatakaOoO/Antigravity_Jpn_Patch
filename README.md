# Antigravity Japanese Patch (Antigravity_Jpn_Patch)

このプロジェクトは、Google DeepMindが開発するAIエージェントプラットフォーム「Antigravity」のUIを日本語化するための非公式パッチです。

## 動作確認済みバージョン
- **Antigravity v2.3.1** （※過去の v2.1.4 および v2.2.1 向けパッチャーも同梱しています）
- （※今後のアップデートにより、仕様変更で適用できなくなる可能性があります）

## 特徴
- **自動適用ツール**: お使いのバージョンに対応する `AntigravityJpnPatcher_v[バージョン].exe`（例: `AntigravityJpnPatcher_v2.3.1.exe`）を実行するだけで、ワンクリックで誰でも簡単に日本語化が可能です。
- **外部辞書ファイル**: 翻訳辞書データは `japanese_patch.json` で管理されています。このJSONを更新して再度インストールスクリプトを実行するだけで、常に最新の翻訳を反映できます。
- **プラグインとスキルの翻訳対応**: UIの基本テキストだけでなく、MCPサーバーや各種スキルの説明文の日本語化にも対応しています。

## 導入方法（インストール）
1. このリポジトリから、お使いのAntigravityのバージョンに合ったパッチャー（例: `AntigravityJpnPatcher_v2.3.1.exe`）をダウンロードします。（※GitHubのReleasesから取得してください）
2. Antigravityアプリが起動している場合は、完全に終了します。
3. ダウンロードしたパッチャー（例: `AntigravityJpnPatcher_v2.3.1.exe`）をダブルクリックして実行します。
4. 黒いコマンドプロンプト画面が立ち上がり、自動でバックアップとパッチ適用が行われます。
5. 「日本語化パッチが正常に適用されました！」と表示され、画面が閉じたら完了です。
6. Antigravityを起動すると、UIが日本語化されています。

## 元の英語版に戻す方法
**`uninstall.ps1`** を右クリックして「PowerShell で実行」を選択してください。自動的にバックアップから元の状態に復元されます。

## 翻訳の追加・修正
1. `japanese_patch.json` をテキストエディタで開き、翻訳を追加・修正します。
2. 再度、対応するパッチャー（例: `AntigravityJpnPatcher_v2.3.1.exe`）をダブルクリックして実行すると、変更がアプリに反映されます。
